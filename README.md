# Project: MPC Prestate
This project aims to discover the 'prestate' phase.
'Prestate' is defined as 4-6s pre-stimulus phase before the exact physical heat stimulus.
## Directory Structure
* '/mpc_wani': mpc wani dataset
* '/mpc_100': mpc 100 dataset
* '/scripts': all scripts are collected
* '/results': all results are collected (including .nii files)
* '/figures': all figures are collected
